# 咕宝聚合指南 — 设计文档

## 概述

"咕宝聚合指南"是一个资源导航网站，整理各品类优质网站，用户点击即可跳转。目标用户是寻找影视、二次元、音乐、阅读、游戏、娱乐、工具等各类资源的人群。

## 技术架构

- **前端**: 原生 HTML/CSS/JS，无框架，无构建步骤
- **数据库**: Supabase 免费云数据库（PostgreSQL）
- **部署**: GitHub Pages
- **认证**: 管理页使用简单密码保护（密码存储在 Supabase 中验证）

```
GitHub Pages (静态文件)
    ├── index.html        → 主页
    ├── admin.html        → 管理后台
    ├── css/style.css
    └── js/
        ├── config.js     → Supabase 连接配置
        ├── main.js       → 主页逻辑
        └── admin.js      → 管理逻辑

Supabase (云数据库)
    ├── categories 表
    └── links 表
```

## 数据库设计

### categories 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int8 (PK, 自增) | 分类 ID |
| name | text (NOT NULL) | 分类名称 |
| icon | text | emoji 图标 |
| sort_order | int | 排序顺序，小的在前 |

### links 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | int8 (PK, 自增) | 链接 ID |
| category_id | int8 (FK → categories.id) | 所属分类 |
| name | text (NOT NULL) | 网站名称 |
| url | text (NOT NULL) | 网站地址 |
| description | text | 简短描述 |
| sort_order | int | 排序顺序 |

## 页面设计

### 主页 (index.html)

**布局结构（从上到下）：**

1. **Header** — 左侧 Logo"✨ 咕宝聚合指南"，右侧简洁导航（可选"关于"链接）
2. **搜索区** — 居中搜索栏，圆角 pill 样式，placeholder"搜索网站名称或关键词..."，输入时实时过滤下方链接
3. **分类标签** — 一行分类 Tab（从 categories 表加载），选中态高亮（蓝紫背景 + 白字），未选中为灰色
4. **链接列表区** — 选中分类后展示该分类下所有链接。每个链接为一张白色卡片：
   - 左侧 emoji 图标
   - 中间网站名称 + 简短描述
   - 点击整张卡片 → 新标签页打开目标网站
   - 卡片左侧有淡色边框装饰

**初始状态**: 默认选中第一个分类，展示其链接。

**搜索行为**: 输入关键词时，切换为"全局搜索模式"，跨所有分类过滤链接，隐藏分类 Tab 切换，只显示匹配结果。

### 管理后台 (admin.html)

**密码门**: 进入页面先显示密码输入框，输入正确密码后进入管理界面。密码存储在 Supabase 的配置中。

**管理界面（侧边栏布局）：**

- **左侧栏** — 分类列表
  - 每个分类项：图标 + 名称 + 链接数
  - 底部"+ 添加分类"按钮
  - 每个分类项右侧有编辑和删除图标（hover 显示）
  - 点击分类选中，右侧加载该分类的链接

- **右侧主区域** — 当前选中分类的链接列表
  - 顶部显示分类名称 + 链接数
  - 链接列表，每行：网站名 + URL + 编辑/删除按钮
  - 底部"+ 添加链接"按钮
  - 添加/编辑链接：弹出 modal，字段：名称、URL、描述
  - 添加/编辑分类：弹出 modal，字段：名称、emoji 图标

## 配色方案

- **背景渐变**: `linear-gradient(135deg, #f0f4ff 0%, #f8f6ff 50%, #eef3fa 100%)`
- **卡片背景**: 白色 `#ffffff`，带微弱阴影
- **主色调（强调色）**: `#5a78dc`（蓝紫），用于选中态、按钮、链接高亮
- **文字主色**: `#3a3d66`（深蓝灰）
- **文字辅色**: `#8899bb`（灰蓝）
- **文字弱色**: `#aabbcc`
- **卡片阴影**: `0 1px 8px rgba(0,0,0,0.04)`

## 功能清单

| 功能 | 说明 |
|------|------|
| 分类展示 | 以图标网格展示所有分类 |
| 分类切换 | Tab 切换选中的分类 |
| 链接展示 | 当前分类下所有链接卡片列表 |
| 实时搜索 | 输入关键词过滤链接（跨分类） |
| 外部跳转 | 点击链接卡片在新标签页打开目标网站 |
| 管理登录 | 密码保护进入管理页 |
| 分类 CRUD | 管理页中增删改分类 |
| 链接 CRUD | 管理页中增删改链接 |

## 非功能需求

- 纯静态文件，直接部署到 GitHub Pages
- 首次加载从 Supabase 读取数据，展示速度快
- 响应式设计：桌面端和移动端均可正常使用
- 搜索过滤在客户端完成（数据量小，无需服务端搜索）
